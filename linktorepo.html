puz·link

If you are participating in the MIT Mystery Hunt and you're not on team Control Group, please don't use this copy of puz.link during the Hunt for solving Hunt puzzles.

The reason is that I may need to debug the tool, which involves looking at web logs, and I don't want to accidentally see any solutions or clues my team has not unlocked.

Please use the github repository, linked below. There are instructions in the README to get your own version of puz.link set up in a few minutes. If you have trouble setting it up, please contact me at cosmologicon@gmail.com. Good luck!
